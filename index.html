Warlords

A pair project building a 4 player version of the atari 2600 game warlords in JavaScript using the HTML5 canvas.

My focus was the game logic for the movement and collision detection on the paddles, and pulling all of the logic together into the game flow.

  • Stack: JavaScript, Webpack
