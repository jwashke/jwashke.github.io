Commissionrableio // Ruby on Rails, JavaScript, JQuery

A pair project built for Trelora, a flat free real estate company. The purpose was to build a website where users can view real estate commissions on homes for sale in the Denver area. Upon creating account, users can view a google map of the Denver area with markers representing houses for sale. Clicking on a marker reveals information about the house with a link to a show page where the user can see more information.

My focus on this project was creating an api endpoint for the homes data, and creating the google maps page to consume that API and draw markers on the map based on the data received, along with filtering those markers using JQuery. I also created a background worker to handle processing an uploaded CSV file for admin users

Learning Focus:

JavaScript AJAX Google Maps JavaScript SDK Sidekiq/Redis

Github Heroku
